國防醫學院
案名：108年度國防醫學院全球資訊網站再造案
108年1月1日起至108年12月31日止
專案說明:
  • 依據行政院「政府網站改善計畫」政策推動，並參照政府網站版型相關規範，國防部全球資訊網網站改版為RWD響應式網頁設計
  • 國防部政務辦公室軍網網站維運建置
  • 國防部網站評鑑服務：聘請外部網站專家學者舉辦網站評鑑會議，並彙整外部評鑑委員撰寫評鑑建議表、評分表等相關紀錄
  • 舉辦全國軍單位系統管理人員教育訓練課程內容為含後台管理系統功能操作訓練、網站管理維運實務
  • 資訊安全管理系統(ISMS)暨個資防護管理系統(PIMS)制度管理維護
  • 資訊安全維護工作：配合國防部年度演習及資安通報執行相關資安設定，並提供國防部適當建議
開發環境及工具說明:
  • RWD響應式網頁設計：Boostrap開發
  • 網站開發語言：HTML5、C#、CSS3、.NET...
  • 作業系統：WIN Server 2012 R2
  • 資料庫：MS SQL Server 2014
